In the UI, type:
0 for Voltorb
1, 2, or 3 for their respective cards
Any other number for Voltorb or 1, and it's uncertain which
Blank for unknown

In the coloured grid, the top number is the % chance that it IS NOT 0, and the bottom number is the % chance that it IS a 2 or 3.

The clue with the largest number of unique lines is (8, 1), with 95 possible lines of 5 cards that match.
